# 用語集

## ドメイン用語

| 日本語             | 英語(システム内部用) | 説明                                                     |
| ------------------ | -------------------- | -------------------------------------------------------- |
| 研究会             | Circle               | ある程度決まったメンバーで継続的に開催される将棋の集まり |
| 開催回             | CircleSession        | 研究会に属する 1 回分の開催                              |
| 対局結果           | Match                | 1 局 = 1 件の対局結果                                    |
| 対局結果の編集履歴 | MatchHistory         | 対局結果の作成・修正・削除の履歴（スナップショット保持） |

## 権限・ロール

| 日本語             | 英語(システム内部用) | 説明                                        |
| ------------------ | -------------------- | ------------------------------------------- |
| 研究会オーナー     | CircleOwner          | 研究会の最上位ロール（必ず 1 人、移譲可能） |
| 研究会マネージャー | CircleManager        | 研究会の運営を補助するロール                |
| 研究会メンバー     | CircleMember         | 研究会の基本ロール                          |
| 開催回オーナー     | CircleSessionOwner   | 開催回の最上位ロール（必ず 1 人、移譲可能） |
| 開催回マネージャー | CircleSessionManager | 開催回の運営を補助するロール                |
| 開催回メンバー     | CircleSessionMember  | 開催回の基本ロール                          |

## 実装上の関連モデル

| 日本語     | 英語(システム内部用)    | 説明                                        |
| ---------- | ----------------------- | ------------------------------------------- |
| 研究会参加 | CircleMembership        | ユーザーと研究会の関連（ロールを 1 つ保持） |
| 開催回参加 | CircleSessionMembership | ユーザーと開催回の関連（ロールを 1 つ保持） |
