# ユースケース定義

## システムの目的

将棋の研究会における参加者と対局結果を記録・閲覧する。

## ユースケース

### 研究会

- 研究会を作成できる
- 研究会の詳細情報（名称など）を編集できる
- 研究会を削除できる
- 研究会の参加者を閲覧できる
- 研究会に参加者を追加できる
- 研究会参加者のロールを変更できる
- 研究会オーナーを移譲できる

### 開催回

- 開催回を作成できる
- 開催回を複製して作成できる
- 開催回の詳細情報（日付・場所など）を編集できる
- 開催回を削除できる
- 開催回を閲覧できる
- 開催回の参加者を閲覧できる
- 開催回に参加者を追加できる
- 開催回参加者を参加取消（除外）できる（対局結果に登場している参加者は除外不可）
- 開催回参加者のロールを変更できる
- 開催回オーナーを移譲できる

### ユーザー

- ユーザーは参加している研究会の一覧を閲覧できる

### 対局結果

- 開催回内の対局結果を記録できる
- 対局結果を閲覧できる
- 対局結果を修正・削除できる
- 対局結果の編集履歴を閲覧できる

## やらないこと（スコープ外）

- 棋譜の保存
- 手合割・持ち時間の管理
- レーティング計算
- 勝敗集計のリアルタイム更新
- 研究会の検索
- 招待の承認フロー
